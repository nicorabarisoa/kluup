# Kluup — Questions v1
> 120 questions | 4 thèmes | 3 types | 3 niveaux | Mis à jour 04/06/2026

---

## Format
- **Type A** — Désignation : "Qui dans le groupe est le plus susceptible de..." → le groupe vote pour désigner quelqu'un d'autre
- **Type B** — Confession : "Qui parmi vous a déjà..." → chacun se désigne soi-même
- **Type C** — Question ouverte → volontariat d'abord, vote si personne
- **Niveau 1** → léger / **Niveau 2** → personnel / **Niveau 3** → intense (relatif au thème)
